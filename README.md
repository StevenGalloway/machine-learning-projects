# Machine Learning Lab

This repository demonstrates practical machine learning across the full lifecycle:
from problem framing to production considerations.

---

## Areas of Focus
- Feature engineering & data leakage prevention
- Model selection & evaluation
- Experiment reproducibility
- Error analysis & iteration
- Drift & retraining strategy

---

## Project Index

| Area | Example |
|-----|---------|
| Classification | Fraud / risk scoring |
| Regression | Demand & price prediction |
| NLP | Text classification |
| Time Series | Forecasting |
| Clustering | Customer segmentation |

---

## Structure
- `case-studies/` – end-to-end ML scenarios
- `src/` – reusable pipelines
- `configs/` – experiment configs
- `notebooks/` – EDA
- `tests/` – validation & correctness
